# 位运算

## 异或（^）

- 相同为0，不同为1
  - 0 ^ 1 = 1 、1 ^ 0 = 1
  - 0 ^ 0 = 0、1 ^ 1 = 0
  - 0 ^ N = N、N ^ N = 0
  - a ^ b = b ^ a、a ^ b ^ c = a ^ (b ^ c)
- 无进位相加
- 可以用来交换两数，速度快，前提a和b指向的内存是独立的区域
  - a = a ^ b
  - b = a ^ b
  - a = a ^ b
- 如果a和b指向的内存是同一块区域，则会把值抹成0

## 与（&） 

- 相同为1，不同为0
- 可以用做取模运算
- 把一个不等于0的数最右侧的1提取出来
  - A = a & ~a + 1

## 乘除 / 左移右移

- 左乘2 -----------   <<1    左移一位等于乘二
- 右除2 -----------   >>1    右移一位等于除二

# 正则

- `()` ：代表分组
- `(.)\1` ：将相同内容分组
  - "aaaabbbbcccc" -> ["aaaa","bbbb","cccc"]
- `(.)\其他数字` 
  - "aaaabbbbcccc" -> ["a","a","a","a","b","b","b","b","c","c","c","c"]
- `$1`
  - 匹配分组中第一个分组 即 "aaaa"
- `+` ：出现1次以上
